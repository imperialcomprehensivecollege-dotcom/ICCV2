🎓 Primary School CMS Installation Guide

Prerequisites

  • PHP 8.2 or higher
  • MySQL 8.0 or higher
  • Composer
  • Node.js 18+ (for asset compilation)

Installation Steps

1. Install Dependencies

composer install

2. Environment Setup

cp .env.example .env

Edit .env and configure:

  • APP_KEY - Run: php artisan key:generate
  • DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD
  • MAIL_* - Email configuration

3. Install Node Dependencies

npm install

4. Build Assets

npm run build

5. Create Database

php artisan migrate

6. Seed Demo Data

php artisan db:seed

7. Link Storage

php artisan storage:link

8. Create Admin User (Optional)

php artisan tinker
App\Models\User::factory()->create(['name' => 'Admin', 'email' => 'admin@example.com'])->assignRole('super-admin')

9. Start Development Server

php artisan serve

Access the site at http://localhost:8000

📝 Development Tips

  • Use php artisan tinker for quick database queries
  • Run npm run dev for Tailwind CSS watch mode
  • Clear caches with php artisan optimize:clear
  • Check migrations: php artisan migrate:status

⚠️ Security Reminders

  • Change default passwords immediately
  • Set APP_DEBUG=false in production
  • Use environment variables for sensitive data
  • Configure CORS if needed
  • Enable HTTPS in production

Available Artisan Commands

# Database
php artisan migrate              # Run migrations
php artisan migrate:rollback     # Rollback migrations
php artisan db:seed              # Run seeders
php artisan db:wipe              # Clear database

# Cache & Optimization
php artisan optimize             # Optimize application
php artisan optimize:clear       # Clear optimizations
php artisan cache:clear          # Clear cache
php artisan config:clear         # Clear config cache

# Users & Permissions
php artisan tinker               # Interactive shell
php artisan make:user            # Create new user

# Filament
php artisan filament:install     # Install Filament

Troubleshooting

Database Connection Error

Ensure MySQL is running and credentials in .env are correct:

php artisan db:table users

Migration Failed

Rollback and try again:

php artisan migrate:rollback
php artisan migrate

Storage Permission Issues

chmod -R 775 storage bootstrap/cache

Asset Issues

npm run build
php artisan optimize:clear
